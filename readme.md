=============
<h1>취지</h1>
소셜미디어에 존재하는 대나무숲(익명으로 전해드립니다.)들은 대부분 관리자의 의해 익명글이 올라가고 있음.
하지만 관리자가 특정 경우에 의해 익명성을 제대로 지키지 않을 수도 있음.

이를 보안하기 위해 관리자라는 역할을 봇이 처리함으로 써 누가 올렸는지 익명성을 보장 할 수 있음.
